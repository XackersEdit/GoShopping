Follow the steps to run the web application:

1) From www.python.org download the python ( > 3.9 version)
2) Delete the venv folder in the project!
3) Download any IDLE and run following code one by one in terminal:
    `python -m venv venv`
    `.\venv\scripts\activate`
    `pip install -r requirements.txt`
    `python manage.py runserver`
4)  When you run the last code you have to see the following outcome:
`
